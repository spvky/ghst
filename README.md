# Ghst

This is my (spvky) personal renderer, anyone is free to use, just expect rapid changes if you do as my plan is to evlolve this as I learn game-dev in odin and use this for various projects, there are comments so that when I go back and change things I understand the changes I'm making, if those help you that's awesome

## Dependencies
This repo assumes that you have downloaded and built sokol, and have the sokol-odin bindings located at `<ODIN-PATH>/shared/sokol`
